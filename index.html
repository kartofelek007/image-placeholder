Parameters:

  • http://pokaimg.me/200x400 set specific size
  • http://pokaimg.me/200x400?bg=red http://pokaimg.me/200x400?bg=ff0000 http://pokaimg.me/200x400?bg=f00 set background color. Color can be given as a html color name, or in hexadecimal form - 3 or 6 characters (without #)
  • http://pokaimg.me/200x400?c=red http://pokaimg.me/200x400?c=ff0000 http://pokaimg.me/200x400?c=f00 Text color. Color can be given as a color name, or in hexadecimal form - 3 or 6 characters (without #)
  • http://pokaimg.me/200x400?text=Lorem set text to "Lorem"
  • http://pokaimg.me/200x400?text= set text to empty (without dimension on image)
  • http://pokaimg.me/200x400?gray=1&text=lorem set grayscale mode
  • http://pokaimg.me/200x400/blur http://pokaimg.me/200x400/luxury?text=lorem&gray=1 generate image from specyfic category
  • http://pokaimg.me/200x400?cross=1 generate cross lines on photo
  • http://pokaimg.me/200x400?id=3 show specyfic image

You can mix all parameters (look below for examples).
