Parameters:

  • https://pokaimg.me/200x400 set specific size. Minimum size is 10x10, maximum is 1000x1000
  • https://pokaimg.me/200x400?bg=red https://pokaimg.me/200x400?bg=ff0000 https://pokaimg.me/200x400?bg=f00 set background color. Color can be given as a html color name, or in hexadecimal form - 3 or 6 characters (without #)
  • https://pokaimg.me/200x400?c=red https://pokaimg.me/200x400?c=ff0000 https://pokaimg.me/200x400?c=f00 Text color. Color can be given as a color name, or in hexadecimal form - 3 or 6 characters (without #)
  • https://pokaimg.me/200x400?text=Lorem set text to "Lorem"
  • https://pokaimg.me/200x400?text= https://pokaimg.me/200x400?notext&bg=red set text to empty (without dimension on image)
  • https://pokaimg.me/200x400?gray&text=lorem set grayscale mode
  • https://pokaimg.me/200x400/blob https://pokaimg.me/200x400/luxury?text=lorem&gray=1 generate image from specific category
  • https://pokaimg.me/200x400?lines generate cross lines on photo
  • https://pokaimg.me/200x400?id=3 show specific image

You can mix all parameters (look below for examples).
